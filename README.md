# 🏀 NBA Player Performance Analysis

A complete machine learning project analyzing NBA player performance using real-world game data from the 2024–2025 season.

---

## 🧠 Overview
This project applies multiple machine learning techniques to understand player performance, predict outcomes, and extract meaningful insights from basketball data.

The project follows a full ML pipeline:
- Data cleaning and preprocessing  
- Feature engineering  
- Model training and evaluation  
- Insight generation  

---

## 🚀 Features
- 📈 Regression: Predict player points (PTS)  
- 🧠 Clustering: Identify player playing styles  
- 🎯 Classification: Predict game outcomes (Win/Loss)  
- ⭐ Custom Analysis: Player efficiency metrics  

---

## 🛠️ Technologies Used
- Python  
- Pandas  
- NumPy  
- Scikit-learn  
- Matplotlib  
- SciPy  

---

## 📊 Dataset
- NBA 2024–2025 player statistics  
- Source: Kaggle  

Files used:
- database_24_25.csv  
- cleaned_data.csv  

---

## 🧪 Machine Learning Tasks

### 1️⃣ Regression
- Linear Regression, Random Forest, SVR  
- Goal: Predict player points  

### 2️⃣ Clustering
- K-Means, Hierarchical Clustering  
- Goal: Group players by style  

### 3️⃣ Classification
- Logistic Regression, Random Forest, SVM  
- Goal: Predict win/loss  

### ⭐ Player Efficiency Analysis
- Custom metrics beyond traditional stats  

---

## 📈 Results
- SVR performed best for regression  
- K-Means identified player roles  
- Logistic Regression achieved best ROC-AUC  
- Classification performance was moderate
