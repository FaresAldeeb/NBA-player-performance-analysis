# 🏀 NBA Player Performance Analysis

A machine learning project analyzing NBA player performance using real-world game data from the 2024–2025 season.

---

## 🧠 Overview
This project applies multiple machine learning techniques to understand player performance, predict outcomes, and extract meaningful insights from basketball data.

---

## 🚀 Features
- 📈 Regression: Predict player points (PTS)  
- 🧠 Clustering: Identify player playing styles  
- 🎯 Classification: Predict game outcomes (Win/Loss)  
- ⭐ Custom player efficiency analysis  

---

## 🛠️ Technologies Used
- Python  
- Pandas, NumPy  
- Scikit-learn  
- Matplotlib  
- SciPy  

---

## 📊 Dataset
- NBA 2024–2025 player statistics  
- Source: Kaggle  

---

## 🧪 Machine Learning Tasks

### 1️⃣ Regression
- Models: Linear Regression, Random Forest, SVR  
- Goal: Predict player points  

### 2️⃣ Clustering
- Models: K-Means, Hierarchical  
- Goal: Group players by style  

### 3️⃣ Classification
- Models: Logistic Regression, Random Forest, SVM  
- Goal: Predict win/loss  

---

## 📈 Results
- SVR achieved best regression performance  
- K-Means successfully grouped player roles  
- Logistic Regression achieved best classification ROC-AUC
